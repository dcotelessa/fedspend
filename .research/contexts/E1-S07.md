## Notes
HealthController injects DataSource (from typeorm) to verify the database connection is active. Return { status:'ok', timestamp: new Date().toISOString(), database: ds.isInitialized ? 'connected' : 'disconnected' }. The spec must mock DataSource and use a table of {isInitialized, expected database field}. ValidationPipe config goes in main.ts bootstrap() — do NOT register it as an APP_PIPE provider in AppModule (keeps the pipe at the HTTP boundary, matching AGENTS.md Skill 1).
