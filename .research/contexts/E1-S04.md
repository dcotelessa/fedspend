## Notes
Per resolution: sync module is module + service only (NO controller) — sync API endpoints belong to Epic 2. Per resolution: health module is module + controller only (NO service). Per resolution: empty dto/ directories must have a .gitkeep so git tracks them. Controllers and services are EMPTY SCAFFOLDS in this story — just the class with @Controller()/@Injectable() decorator and an empty body. No endpoints, no methods. They get implemented in Epic 3 (query API). The modules.spec.ts must assert each module class exists, is decorated with @Module, and (where applicable) registers its controller/provider.
