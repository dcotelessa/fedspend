## Notes
Use the Angular 20 standalone API (default — no NgModules). ng new flags: --skip-git (avoid a nested repo inside the build worktree), --skip-install (this is a pnpm workspace; register 'frontend' in pnpm-workspace.yaml then run `pnpm install` at the REPO ROOT so the single root pnpm-lock.yaml covers both backend and frontend). Add runtime deps to the frontend workspace package with `pnpm add --filter frontend @angular/material @angular/cdk ng2-charts chart.js`, and devDeps with `pnpm add -D --filter frontend jest-preset-angular jest @types/jest jsdom` (jest-preset-angular@latest requires Angular >=20, jest ^30, jsdom ^26). Remove karma.conf.js and the karma-related devDeps (karma, karma-chrome-launcher, karma-coverage, karma-jasmine, jasmine-core) and remove the 'test' builder entry in angular.json; set the frontend package.json 'test' script to jest. jest.config.js: preset 'jest-preset-angular', setupFilesAfterEnv ['<rootDir>/setup-jest.ts'], testEnvironment 'jsdom'. setup-jest.ts: `import 'jest-preset-angular/setup-jest';`. tsconfig.spec.json compilerOptions.types: ['jest','node']. Add to frontend/tsconfig.json compilerOptions.paths: "@shared/*": ["../shared/*"] (no baseUrl needed under TS5). app.spec.ts asserts the root AppComponent compiles. The whole frontend/ tree is committed via scope.commitAllUnder (you create it with the CLI); the root pnpm-lock.yaml is also auto-staged.
